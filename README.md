
# validate-proto-http

Validate gRPC HTTP options mapping the methods to JSON.

### Install

```
curl -sSL https://git.io/fjE3d | sudo bash
```

### Usage

Run the app in the folder where you want to scan recursively all the protos:

```
validate-proto-http
```
